# 活動メンバーについて

ゆかコネNEOを楽しく使いながら、みんなにその使い方や文化を広げてもらおうということで、アンバサダーを任命しています。

今期は、その活動の幅から、エバンジェリスト（ガイド）、アンバサダー、研究生、活動支援者の４つに区分けしています。
