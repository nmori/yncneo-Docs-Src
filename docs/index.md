# さぁ、言葉のその先へ！

![ゆかコネ - リアルタイム字幕・翻訳・連携ツール](images/title02.png)

**ゆかコネは声を字幕にして、多言語に翻訳できる配信支援ツール。**  
難しい設定は必要なく、声を話すだけでリアルタイムに字幕にできます！
